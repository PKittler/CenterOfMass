# CenterOfMass
Tool to find the center of mass in a 3-axis-system to analyze the structure and concentration of groups.

## Center of Mass
The position of the center of mass is declared by a vector r.

r = (r_1 * m_1 + r_2 * m_2 + ... + r_n * m_n) / m
